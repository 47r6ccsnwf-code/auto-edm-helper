# 🏗️ 自动 EDM 助手 (Suntec Mailer V2) 用户使用手册

本工具是一款轻量级、不依赖第三方平台的批量冷邮件营销系统。专门为跨境贸易场景设计，支持多账号轮发、个性化模板以及实时看板监控。

## 🌟 核心功能
*   **多账号轮发**：支持 Gmail/Outlook/企业邮，自动轮换发信，保护账号权重。
*   **瑞士风看板**：内置极简风格的 Web 仪表盘，支持实时查看发信进度及回信。
*   **个性化变量**：自动提取 CSV 线索中的公司名、国家、联系人姓名进行内容填充。
*   **防封号策略**：内置随机停顿（45-180秒）与每日限额，最大限度模拟人工操作。
*   **断点续发**：利用本地 SQLite 数据库记录状态，任务中断后重启可自动跳过已发客户。

---

## 🛠️ 快速上手

### 1. 环境准备
确保您的电脑已安装 **Python 3.10+**。

### 2. 安装依赖
在终端中进入项目目录，执行：
```bash
cd mailer
pip install -r requirements.txt
```

### 3. 配置邮箱 (重要)
打开 `config.yaml` 文件，填入您的发信信息。

#### Gmail 用户注意：
1. 必须开启 Google 账号的 **「两步验证」**。
2. 必须生成 **「应用专用密码」**（App Password），并填入 `password` 字段。
3. 请勿直接使用登录密码。

### 4. 运行系统

#### A. 启动 Web 看板（推荐）
```bash
python webapp.py
```
👉 访问：`http://127.0.0.1:5005`
在看板中您可以：
*   预览每一封个性化邮件。
*   一键启动后台发送。
*   刷新并查看邮件里的最新回复。

#### B. 使用命令行工具
*   **预览邮件**：`python cli.py preview --limit 10`
*   **正式发送**：`python cli.py send --limit 20`
*   **查看状态**：`python cli.py status`

---

## 📂 目录结构说明
*   `engine.py`: 系统核心引擎，负责发信逻辑、节流与数据库。
*   `webapp.py`: 仪表盘后台。
*   `config.yaml`: 所有的发信配置、限额、线索文件路径都在这里改。
*   `templates/`: 存放邮件文案，支持多语言（EN/FR/AR）。
*   `leads/`: 存放您的 CSV 线索表。

---

## ⚠️ 安全建议
1.  **慢就是快**：系统默认设置的日发信量为 30-50 封，这是为了防止您的发信域名被判定为垃圾邮件。
2.  **线索核验**：发信前请确保 `MASTER_leads_verified.csv` 中的数据准确。
3.  **SPF/DKIM**：如果您使用企业邮箱域名，请务必在域名解析中配置 SPF 和 DKIM 记录，否则到达率会极低。
